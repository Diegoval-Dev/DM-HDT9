En esta seccin se desarrolla el primer modelo de redes neuronales artificiales (RNA) para la tarea de clasificacin del precio de las viviendas en tres categoras: econmica, intermedia y cara.

Se implementar una red neuronal con una topologa bsica que incluye dos capas ocultas y una funcin de activacin relu para cada capa. Como funcin de activacin final se utilizar softmax, ya que se trata de un problema de clasificacin multiclase.

Este modelo busca establecer un punto de partida para comparar con una segunda arquitectura ms compleja. Evaluaremos su desempeo con mtricas comunes como la matriz de confusin y las proporciones de aciertos en cada clase. El primer modelo de red neuronal artificial implementado utiliz dos capas ocultas con funciones de activacin relu, y una capa de salida con activacin softmax, adecuada para un problema de clasificacin multiclase.

Durante el proceso de entrenamiento de 50 pocas, el modelo alcanz una exactitud de entrenamiento del 100%. Sin embargo, la exactitud en el conjunto de prueba se estabiliz en aproximadamente un 77.7% a 78.1%, mientras que la funcin de prdida (val_loss) mostr un incremento progresivo en las ltimas pocas.

Este comportamiento sugiere la posible presencia de sobreajuste, ya que el modelo aprende demasiado bien los datos de entrenamiento pero no generaliza igual de bien a datos no vistos. A pesar de ello, el rendimiento alcanzado es competitivo frente a modelos de clasificacin previamente implementados como KNN, SVM o rboles de Decisin.

Anlisis de la Matriz de Confusin del Primer Modelo de Clasificacin

El primer modelo de red neuronal fue evaluado mediante la matriz de confusin, la cual revela los aciertos y errores de clasificacin para cada categora del precio de las viviendas.

  • Categora "Cara":

    • 87 instancias fueron correctamente clasificadas.
    • 10 instancias fueron clasificadas incorrectamente como "Intermedia".
    • No hubo confusin con la categora "Econmica".

  • Categora "Econmica":

    • 76 instancias fueron clasificadas correctamente.
    • 19 instancias fueron confundidas como "Intermedia".
    • 2 instancias fueron clasificadas errneamente como "Cara".

  • Categora "Intermedia":

    • 65 instancias fueron clasificadas correctamente.
    • 18 instancias fueron clasificadas errneamente como "Cara".
    • 15 instancias fueron clasificadas errneamente como "Econmica".

En general, el modelo mostr una mayor precisin para la clase "Cara" y un desempeo razonable en "Econmica", aunque present ms errores de confusin en la clase "Intermedia", donde las predicciones fueron ms dispersas.

Estos resultados son consistentes con la exactitud general observada durante el entrenamiento y validacin. En particular, las categoras "Econmica" e "Intermedia" parecen tener fronteras ms difusas en el espacio de atributos, lo cual es razonable considerando la complejidad inherente al mercado inmobiliario.

def plot_learning_curves(history, model_name):
    acc = history.history['accuracy']
